h2. Запрос

*Метод*: GET
*URI*: /api/client/mobile/2.0/tariffs

Важно наличие заголовка @X-Hive-GPS-Position@ в запросе для точного определения службы такси, обслуживающей регион клиента.

----

h2. Ответ

*Content-Type*: application/json; charset=utf-8
*Body*: Объект Service.

h3. Service

|_.Name|_.Type|_.Required|_.Description|
|kind|string ServiceKind|true|Тип объекта сервиса|
|tariffs|array object Tariff|true, if kind=@service@|Список тарифов|
|dispatcherCall|object DispatcherCall|true, if kind=@service@|Параметры для связи с диспетчерской|
|address|[[object Address|Адресные-объекты#object-address]]|true, if kind=@service@|Населенный пункт, к которому привязана служба такси|
|message|string|true, if kind=@stub@|Сообщение о недоступности сервиса|

h3. ServiceKind

|_.Value|_.Description|
|service|Настройки сервиса|
|stub|Заглушка|

h3. Tariff

|_.Name|_.Type|_.Required|_.Description|
|name|string|true|Наименование тарифа|
|options|array [[Option|Получение-списка-тарифов#Option]]|true|Список опций|

h3. Option

|_.Name|_.Type|_.Required|_.Description|
|id|number|true|ID опции|
|name|string|true|Имя опции|
|type|string [[OptionValueType|Получение-списка-тарифов#OptionValueType]]|true|Тип стоимости|
|value|number|true|Стоимость опции|

h3. OptionValueType

@fixed@ - Фиксированная стоимость
@percent@ - Процент от стоимости заказа

h3. DispatcherCall

|_.Name|_.Type|_.Required|_.Description|
|allow|string AllowCall|true|Можно ли звонить диспетчеру и как|
|number|string|false|Телефон диспетчерской|

h3. AllowCall

@direct@ - Можно позвонить в диспетчерскую напрямую
@via server@ - Можно заказать связь через сервер
@no@ - Связаться с диспетчером нельзя
