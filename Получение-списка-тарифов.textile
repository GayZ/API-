h2. Запрос

*Метод*: GET
*URI*: /api/client/mobile/2.0/tariffs

h3. URI параметры

|_.Name|_.Type|_.Required|_.Description|
|lon|number|false|Долгота|
|lat|number|false|Широта|

----

h2. Ответ

*Content-Type*: application/json; charset=utf-8
*Body*: Объект Service.

h3. Service

|_.Name|_.Type|_.Required|_.Description|
|kind|string ServiceKind|true|Тип объекта сервиса|
|tariffs|array object Tariff|true, if kind=@service@|Список тарифов|
|message|string|true, if kind=@stub@|Сообщение о недоступности сервиса|

h3. ServiceKind

|_.Value|_.Description|
|service|Настройки сервиса|
|stub|Заглушка|

h3. Tariff

|_.Name|_.Type|_.Required|_.Description|
|name|string|true|Наименование тарифа|
|address|object [[Address|Адресные-объекты#Address]]|true|Населенный пункт, к которому привязан тариф|
|options|array [[Option|Получение-списка-тарифов#Option]]|true|Список опций|

h3. Option

|_.Name|_.Type|_.Required|_.Description|
|id|number|true|ID опции|
|name|string|true|Имя опции|
|type|string [[OptionValueType|Получение-списка-тарифов#OptionValueType]]|true|Тип стоимости|
|value|number|true|Стоимость опции|

h3. OptionValueType

@fixed@ - Фиксированная стоимость
@percent@ - Процент от стоимости заказа
