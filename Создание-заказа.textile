Для выполнения данного запроса, пользователь должен быть [[авторизован|HMAC-авторизация]].

----

h2. Запрос

*Метод*: POST
*URI*: /api/client/mobile/2.1/orders
*Content-Type*: application/json; charset=utf-8
*Body*: Объект CreateRequest

h3. CreateRequest

|_.Name|_.Type|_.Required|_.Description|
|tariff|string|true|Наименование тарифа|
|options|array number|true|Выбранные опции тарифа|
|route|array [[Address|Адресные-объекты#address]]|true|Маршрут|
|phone|string|true|Телефон для связи|
|time|string [[OffsetDateTime|Форматы-даты-и-времени#offsetdatetime]]|false|Время заказа (отсутствует, если на текущее)|
|comment|string|false|Комментарий к заказу|

----

h2. Ответ

*Content-Type*: application/json; charset=utf-8
*Body*: Объект Created.

h3. Created

|_.Name|_.Type|_.Required|_.Description|
|id|number|true|Идентификатор заказа|

----
