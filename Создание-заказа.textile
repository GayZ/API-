Для выполнения данного запроса, пользователь должен быть [[авторизован|HMAC-авторизация]].

----

h2. Запрос

*Метод*: POST
*URI*: /api/client/mobile/3.0/orders
*Content-Type*: application/json; charset=utf-8
*Body*: object CreateRequest

h3. object CreateRequest

|_.Name|_.Type|_.Required|_.Description|
|tariff|string|true|Наименование тарифа|
|options|array number|true|Выбранные опции тарифа|
|route|array object OrderAddress|true|Маршрут|
|time|string [[LocalDateTime|Форматы-даты-и-времени#localdatetime]]|false|Время заказа (отсутствует, если на текущее)|
|comment|string|false|Комментарий к заказу|
|costCorrection|number|false|Корректировка стоимости|
|useBonuses|number|false|Количество бонусов для использования в качестве оплаты|

h3. object OrderAddress

|_.Name|_.Type|_.Required|_.Description|
|address|[[object Address|Адресные-объекты#object-address]]|true|Адрес|
|entrance|string|false|Подъезд|
|flat|string|false|Квартира|
|comment|string|false|Комментарий|

----

h2. Ответ

*Content-Type*: application/json; charset=utf-8
*Body*: Объект Created.

h3. Created

|_.Name|_.Type|_.Required|_.Description|
|id|number|true|Идентификатор заказа|
