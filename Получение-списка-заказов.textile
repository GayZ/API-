Для выполнения данного запроса, пользователь должен быть [[авторизован|HMAC-авторизация]].

----

h2. Запрос

*Метод*: GET
*URI*: /api/client/mobile/1.0/orders

----

h2. Ответ

*Content-Type*: application/json; charset=utf-8
*Body*: Массив объектов ShortOrderInfo.

h3. ShortOrderInfo

|_.Name|_.Type|_.Required|_.Description|
|id|number|true|Идентификатор заказа|
|state|number [[OrderState|Получение-списка-заказов#orderstate]]|true|Статус заказа|
|route|array [[Address|Адресные-объекты#address]]|true|Маршрут заказа|
|time|string [[OffsetDateTime|Форматы-даты-и-времени#offsetdatetime]]|false|Время заказа (отсутствует, если на текущее)|

h3. OrderState

@1@ - Создан
@2@ - Назначен
@3@ - По адресу
@4@ - Выполняется
