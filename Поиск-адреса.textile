Данный запрос служит для поиска адресных объектов по заданному шаблону. 

Результатом выполнения этого запроса могут быть:
 * неполные элементы адресной иерархии (улицы, название населенных пунктов, районов, городов и т.д.) каким-либо образом соответствующих заданным критериям поиска. Такие объекты можно использовать для дальнейшего поиска по
адресному каталогу.
 * полный адресный объект с псевдонимом. Критерию поиска тут будет удовлетворять именно псевдоним адресного объекта. В этих объектах есть координаты, они могут быть элементами маршрута в заказе.

h2. Формат запроса

*Метод*: GET
*URI*: /api/client/mobile/1.0/address/address

h3. Параметры GET запроса

|_.Name|_.Type|_.Required|_.Description|
|pattern|string|true|Аргумент для полнотекстового поиска|
|parent|number|false|ID родительского адресного объекта|

----

h2. Формат ответа

*Content-Type*: application/json; charset=utf-8
*Body*: Массив объектов [[Address|Поиск-адреса#Address]].

h3. Address

|_.Name|_.Type|_.Required|_.Description|
|level|array [[AddressLevel|Поиск-адреса#AddressLevel]]|true|Адрес и его родители (в порядке от потомка к родителю)|
|priority|number|false|Приоритет|
|point|object [[Point|Поиск-адреса#Point]] |false|Адресная точка с координатами|

h3. AddressLevel

|_.Name|_.Type|_.Required|_.Description|
|id|number|true|ID адресного объекта|
|type|number|true|Тип адресного объекта|
|name|string|true|Имя адресного объекта|
|prefix|string|true|Префикс имени адресного объекта|

Типы:
0 - Страна
1 - Регион
2 - Автономный округ
3 - Область
4 - Город
5 - Район города
6 - Населенный пункт
7 - Улица

h3. Point

|_.Name|_.Type|_.Required|_.Description|
|id|number|true|ID адресной точки|
|info|object [[PointInfo|Поиск-адреса#PointInfo]]|true|Опциональные информационные данные по точке|
|coordinates|object "GeoJSON::Point":http://geojson.org/geojson-spec.html#point|false|Координаты|

h3. PointInfo

|_.Name|_.Type|_.Required|_.Description|
|alias|string|false|Псевдоним точки|
|house|string|false|Номер дома, включающий в себя буквы и дроби|
|housing|string|false|Номер корпуса дома|
|building|string|false|Номер строения|
|lit|string|false|Литера|
|porch|string|false|Подъезд|
|flat|string|false|Квартира|

h2. Пример

https://my-taxi.mydomain/api/client/mobile/1.0/address/address?pattern=ленина&parent=37717755
</code></pre>
